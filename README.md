# Adaptive Structure-Aware Repulsion: Guiding Graph Contrastive Learning Beyond Neural Collapse
> ⚠️ **NOTE:** This research is an automatic research using AIRAS.
## Abstract
Graph Contrastive Learning (GCL) has emerged as a powerful paradigm for self-supervised representation learning on graphs. However, its theoretical underpinnings are at a turning point. The prevailing Neural Collapse theory, which posits a Simplex Equiangular Tight Frame (ETF) as the optimal geometry, is being challenged by a more nuanced 'low-rank bias' theory, suggesting that deep models favor more structured, lower-rank solutions. This exposes a critical gap: existing GCL methods are not designed for this new paradigm. This is a challenging problem because current regularizers are either computationally prohibitive for large graphs, scaling quadratically with the number of nodes, or they are inefficient, applying indiscriminate repulsive forces that waste model capacity on trivial negative samples. To bridge this gap, we introduce Adaptive Structure-Aware Repulsion (ASAR), a novel and scalable loss-based regularizer that explicitly operationalizes the low-rank bias theory. ASAR features a dual-adaptive mechanism: it dynamically identifies a set of 'confusable' hard negatives for each anchor based on an adaptive similarity margin, and it applies a localized repulsive force whose strength is modulated by a self-organizing feedback controller that responds to the batch's representation geometry. This makes ASAR a unique, local, and adaptive regularizer with a computational complexity of O(b^2), making it suitable for massive graphs. We designed a comprehensive experimental protocol to verify ASAR's efficacy on large-scale OGB benchmarks, aiming to demonstrate its superiority over standard GCL baselines and validate its theoretical contributions.
