# Meta-BAPL: Meta-Learning Budget-Aware Policies for Adaptive On-Demand Inference
> ⚠️ **NOTE:** This research is an automatic research using AIRAS.
## Abstract
Dynamic neural networks offer a promising path toward efficient inference, yet their practical deployment is hindered by inflexibility. Current approaches are largely bifurcated: static methods require training and maintaining a cumbersome 'model zoo' to serve diverse performance-cost needs, while dynamic methods offer runtime control through indirect and often unpredictable heuristics like confidence thresholds. This creates a critical gap for a single model that can precisely and predictably adapt its behavior to explicit, multi-dimensional system constraints such as latency and energy. We address this challenge by proposing a paradigm shift from instance-reactive control to system-proactive, budget-aware control. Our contribution is Meta-BAPL, a meta-learning framework that trains a single dynamic network to generate an optimal inference policy conditioned on an explicit, multi-dimensional budget vector provided at runtime. The framework incorporates a budget-conditioned meta-policy, a transferable GNN-based cost model for end-to-end training, and a novel smoothness regularizer to ensure predictable generalization to unseen budgets. We verify our approach on the ImageNet benchmark, demonstrating that a single Meta-BAPL model can trace the entire accuracy-cost Pareto frontier, outperforming static model zoos and providing precise, direct control over resource consumption, with budget adherence errors below 5%.
