# Velocity-Optimized Meta-Learning: Directly Targeting Convergence Speed
> ⚠️ **NOTE:** This research is an automatic research using AIRAS.
## Abstract
Gradient-based meta-learning algorithms, such as Model-Agnostic Meta-Learning (MAML), are foundational for few-shot learning but are fundamentally limited by a 'short-horizon bias.' By optimizing for performance after a small, fixed number of adaptation steps, they learn initializations that excel at rapid early gains but often converge to suboptimal local minima, misaligning the meta-objective with the true goal of fast convergence. This is a challenging problem because naively extending the adaptation horizon is computationally prohibitive and can lead to unstable gradients. We introduce Velocity-Optimized Meta-Learning (VeloML), a new framework that directly addresses this objective mismatch. Our primary contribution is a novel meta-objective that acts as a differentiable surrogate for the number of steps required to solve a task, explicitly rewarding initializations that yield the fastest convergence paths. To handle task heterogeneity, we introduce a meta-learned relative thresholding mechanism that allows the model to learn a unified notion of 'solved' across a diverse task distribution. Finally, to ensure computational tractability, we propose a trajectory-aware hybrid-order gradient approximation that strategically modulates gradient fidelity along the adaptation path. We designed a comprehensive experimental protocol on standard few-shot benchmarks, Mini-ImageNet and Tiered-ImageNet, to verify that VeloML learns to solve new tasks in significantly fewer gradient steps, enables principled adaptive computation at inference time, and improves generalization by seeking wider, more stable solution basins.
