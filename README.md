# Mixing-Gated Diffusion: Budgeted, Per-Node, Per-Hop Control for Oversmoothing-Resilient Graph Learning
> ⚠️ **NOTE:** This research is an automatic research using AIRAS.
## Abstract
Oversmoothing arises when cross-group mixing outpaces within-group denoising in graph learning, flattening node representations and hurting accuracy. Non-asymptotic theory shows this trade-off is local across nodes and hops, while recent analyses prove that attention by itself cannot avert exponential loss of expressivity with depth. Existing remedies either apply global filters, normalize post hoc, or inject randomness, and none yields a deterministic, interpretable policy that decides which hops each node should aggregate. We propose MiGAD, a budgeted, compatibility-aware diffusion guard. MiGAD learns soft groups and a row-stochastic group compatibility matrix, estimates per-node, per-hop mixing risk by contrasting local compatible groups with k-hop group diffusion, and enforces a node-specific budget via a differentiable, constrained hop-attention layer. The method plugs into decoupled multi-hop backbones, remains complementary to normalization, and provides deterministic, per-node hop allocations at inference. We specify end-to-end PyTorch/Geometric code, diagnostics, ablations, and rigorous protocols spanning depth robustness, heterophily and corruption, and interpretability. Our provided run establishes an executable environment and training pipeline; quantitative outcomes were not captured in the shared logs, so we report no numbers, while supplying reproducible procedures to test selective long-range diffusion.
