# Learning by Looking Back: Momentum-Based Trajectory Debiasing for Contrastive Learning
> ⚠️ **NOTE:** This research is an automatic research using AIRAS.
## Abstract
Contrastive self-supervised learning thrives on differentiating an anchor from a large set of negative samples. Its performance, however, is critically undermined by the presence of 'false negatives'—semantically similar instances that are inadvertently repelled, leading to a fragmented embedding space. This is particularly challenging because effective learning requires 'hard negatives', which are often visually indistinguishable from false negatives. Existing methods attempt to solve this by using static class priors or indirect heuristics like score variance, but these approaches are either too coarse or rely on ambiguous signals. A stable, true hard negative can appear identical to a false negative under these heuristics. We propose Momentum-based Trajectory Debiasing (MTS), a novel method that identifies false negatives by leveraging a direct and unambiguous signal: the temporal trajectory of similarity scores. By exploiting the inherent temporal gap between a standard online encoder and its slower-moving momentum counterpart, MTS computes a state-free proxy for the similarity gradient. A consistently increasing similarity is a strong indicator of a false negative. This signal is used to dynamically and selectively reduce the repulsive force on suspected false negatives within the contrastive loss. Experiments on the CIFAR-100 fine-grained benchmark show that MTS significantly outperforms standard contrastive learning and existing debiasing techniques, demonstrating its ability to learn more robust and semantically coherent representations.
