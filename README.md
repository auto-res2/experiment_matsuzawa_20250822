# PACER-FL: Auditable Tree-Correlated DP, Residual Updates, and FO-Aware Compression for Cross-Device Federated Learning
> ⚠️ **NOTE:** This research is an automatic research using AIRAS.
## Abstract
Cross-device federated learning requires user-level differential privacy without trusting the server and within tight mobile communication budgets. The prevailing practice—DP-FedAvg with uniform clipping and i.i.d. Gaussian noise—induces clipping bias, accumulates noise as the square root of rounds, and degrades under non-IID data. We present PACER-FL, an auditable framework that co-designs four secure-aggregation–compatible components: client-synthesized tree-correlated Gaussian noise aligned with linear compression; residualized updates with DIANA-style control variates to shrink sensitivity; differentially private participation gating and DP quantile clipping to automate thresholds; and first-order–aware linear sketches that match compression error to the DP noise floor. A unified accountant composes fixed-size subsampling Rényi DP, Laplace AboveThreshold, DP histograms, and DP-FTRL-class correlated Gaussians, while a transparency ledger enables public verification of realized noise. We instantiate PACER-FL on non-IID CIFAR-10 and EMNIST and compare against DP-FedAvg with i.i.d. noise, server-side DP-FTRL, and communication baselines. We release code, an auditor, and a reproducible pipeline; quantitative end metrics are obtained from runs using the provided accounting and logs, enabling verifiable accuracy–privacy–efficiency analyses without unverifiable claims.
