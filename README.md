# Compressive SHAP: Sublinear-Query, Uncertainty-Aware Shapley Explanations via Sparse Walsh–Hadamard Decoding
> ⚠️ **NOTE:** This research is an automatic research using AIRAS.
## Abstract
We study per-instance Shapley explanations under strict API-call budgets and latency constraints, which increasingly govern audits, tracing, and regulated deployment. Black-box estimators such as KernelSHAP and SHAP-IQ typically require at least linear-in-d queries for acceptable accuracy, while model- or gradient-specific accelerations restrict applicability [covert_2020_improving, fumagalli_2023_shap, muschalik_2024_shapiq, yu_2022_linear, chau_2021_rkhs]. We introduce Compressive SHAP (CS-SHAP), a model-agnostic, gradient-free method that reframes local explanation as sparse recovery on the subset lattice. We treat the interventional game v(S)=f(x_S, x̄_{¬S}) as a pseudo-Boolean function, estimate its largest Walsh coefficients via sparse Walsh–Hadamard heavy-hitter decoding, and map them exactly to Shapley values using a closed-form subset–Möbius transform. Under compressibility, CS-SHAP uses O(s·polylog d) masked queries, produces anytime explanations with calibrated confidence intervals via median-of-means and tail-energy probes, and includes an early non-sparsity detector that falls back to sampling [covert_2020_improving, fumagalli_2023_shap]. We validate CS-SHAP on synthetic pseudo-Boolean games and a controlled interaction-discovery task. Pilot results show monotone error reduction with increased budget, accurate recovery of low-order effects that dominate attributions, conservative uncertainty, and gains from order-skewed hashing. We provide implementation-ready protocols and code skeletons to enable full-scale evaluation and integration.
