# Spatial State Space Duality: A Principled 2D Generalization of Mamba-2 for Vision Backbones

## Abstract
State Space Models (SSMs) have emerged as highly efficient architectures for sequence modeling, yet their application to computer vision has been constrained by a fundamental mismatch: retrofitting 1D sequential algorithms to model 2D spatial data. Existing vision SSMs, such as VMamba, rely on heuristic scanning patterns that flatten image data, imposing an artificial causal bias and failing to capture the isotropic nature of visual information. This research addresses this critical gap by introducing Spatial State Space Duality (S4D), a novel operator that algorithmically generalizes the core principles of Mamba-2's State Space Duality (SSD) to two dimensions. Our contribution is a new computational paradigm that shifts from heuristic spatial traversal to principled 2D state propagation. The S4D block decomposes computation into two synergistic steps: a massively parallel intra-chunk 2D mixer to capture local patterns and a novel inter-chunk 2D fused recurrence that natively models spatial dependencies by computing each state as a function of its 2D causal neighbors. We integrate S4D into a hierarchical backbone named Vision Mamba-2 (ViM²). To validate our approach, we outline a comprehensive benchmarking strategy for ViM² on challenging dense prediction tasks, including semantic segmentation on ADE20K and object detection on COCO, against state-of-the-art Transformer, CNN, and prior SSM-based models. We anticipate that ViM² will establish a new state-of-the-art in performance and efficiency, demonstrating the power of a vision backbone that is structurally congruent with images and computationally superior by design.
