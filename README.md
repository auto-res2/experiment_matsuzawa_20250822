# ALM-Cache: Learning an Adaptive Meta-Policy for Compressing Large Language Model KV Caches
> ⚠️ **NOTE:** This research is an automatic research using AIRAS.
## Abstract
The substantial memory footprint of the Key-Value (KV) cache is a primary bottleneck for efficient, long-context inference in Large Language Models (LLMs). Prevailing compression techniques rely on static, manually-tuned heuristics that fail to adapt to the dynamic nature of language generation. These methods commit to a fixed trade-off between competing compression axes—such as recency, saliency, and structural redundancy—which is suboptimal across diverse tasks and different stages of generation. This paper introduces ALM-Cache (Adaptive Layer-wise Meta-Cache), a novel framework that reframes KV cache management as a learned, real-time control problem. We replace static heuristics with a meta-policy, learned via reinforcement learning, that dynamically orchestrates a unified set of compression strategies on a per-layer basis. Our contributions are: first, a unified compression framework that integrates recency, saliency, and layer-merging into a controllable action space; second, a reinforcement learning agent that learns to manipulate these controls to optimize for accuracy, memory, and latency simultaneously; and third, a practical two-stage training regimen with a hybrid proxy reward for tractable learning. We validate our approach through a comprehensive experimental design comparing ALM-Cache against strong static baselines on tasks requiring different context management strategies, including summarization, code generation, and question answering. Our analysis projects that ALM-Cache will establish a new state-of-the-art Pareto frontier, demonstrating superior performance by autonomously discovering and applying near-optimal, context-sensitive compression policies.
