# HiLo‑STeP++: Low‑Rank Stochastic Taylor Probing for High‑Dimensional Diffusions
> ⚠️ **NOTE:** This research is an automatic research using AIRAS.
## Abstract
Estimating weak‑order K≤3 stochastic‑Taylor coefficients for E[g(X_T)] in high‑dimensional Itô/Stratonovich diffusions is central to scalable Greeks, stable neural‑SDE training signals, and weak‑Taylor PDE corrections, yet it is computationally prohibitive and statistically brittle. Dense spatial derivative flows scale as O(d^2–d^3) when σ depends on x, while isolating order‑K terms inflates variance. We introduce HiLo‑STeP++, a Jacobian‑vector‑product (JVP)‑only framework that represents coefficients in a Hall basis of Lie brackets, computes bracket actions via recursive JVPs with randomized contractions, restricts computation to a locally active diffusion subspace, and collapses variance using coefficient‑specific Poisson/BSDE control variates. An occupancy‑aware, multi‑task randomly pivoted Cholesky (RPC) selector allocates evaluations to a handful of high‑leverage time–state nodes, and multilevel Monte Carlo (MLMC) with time randomization couples Brownian paths and probe seeds for end‑to‑end bias–variance control. On a quick GPU run, HiLo‑STeP++ exactly recovers the active rank and reduces a basket‑option Delta variance by 58.25× versus a pathwise baseline at d=100, r=5; synthetic commuting systems exhibit the expected bracket sparsity. A non‑smooth drift test surfaced a strict‑mode autograd issue and suggests straightforward implementation fixes. These results support scalability with effective rank and practical variance reductions for Greeks while preserving coefficient‑level correctness.
